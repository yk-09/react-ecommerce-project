# Kamna Ecommerce Webapp

React + TypeScript frontend built with Vite.

## Requirements

- Node.js 24+
- npm 11+

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
npm run typecheck
npm run format
npm run check
```

## Development

Use `npm run dev` to start the local Vite server. Use `npm run check` before committing to run TypeScript, ESLint, and Prettier checks together.
